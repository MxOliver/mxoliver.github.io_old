Projects

Insomnia Chats

(ReactJS, Firebase, AWS s3, CloudFront)

Insomnia Chats is a React application I created during my Bloc apprenticeship based off an old school chat room interface that uses a Firebase database to store messages and room data. Insomnia chats is currently hosted as a static website through Amazon's s3 service and then deployed through CloudFront for faster delivery and to enable HTTPS.

Vintage Etiquette Twitter Bot

(Python, Twitter API, Heroku, AWS s3)

The Vintage Etiquette Bot is a python-based twitter application I created as a personal project that uses a Markov Chain algorithm as a predictive text generator to create and tweet phrases based on an 1883 Etiquette manual. The application is stored on Amazon's s3 cloud service and deployed through Heroku.

Blues-berry Jams

(ReactJS, AWS s3, CloudFront)

Blues-berry Jams is a React based music streaming application I created as part of my Bloc apprenticeship.. It utilizes the reusability of react components to display album information, and is currently deployed through the AWS CloudFront content delivery network.
